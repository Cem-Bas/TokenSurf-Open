Docs

Get runs flowing into this dashboard in a few lines.

1 · Track

Decorate your agent so each run is captured as a trace — framework-agnostic, and not in your data path.

import tokensurf as ts @ts.track def my_agent(question): ...
2 · Score

Define scorers and run an eval over your cases. See Scorers for the four families.

from tokensurf import evaluate, Dataset from tokensurf.scorers import ExactMatch, ToolCalled report = evaluate( task=my_agent, data=Dataset.from_jsonl("cases.jsonl"), scorers=[ExactMatch(), ToolCalled(name="kb_lookup")], )
3 · Push

Send the run to this server with your project's ingest key. It appears under Projects and Runs, and quality gates evaluate automatically.

tokensurf eval run eval.py \ --server https://your-tokensurf \ --key ts_your_project_key
4 · Gate & alert

In Settings, add a quality gate (e.g. pass-rate ≥ 90%) and a notification channel. A run that breaches a gate shows a red chip and fires your Slack / webhook / email.
