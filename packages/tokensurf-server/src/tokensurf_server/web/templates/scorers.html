Scorers

The four scorer families that grade every agent run. Each score is normalized to 0–1.

LLM-judge flexible

An LLM grades the response against criteria — accuracy, completeness, relevance, helpfulness — on a 1–10 rubric, normalized to 0–1. Provider-agnostic via litellm; bring your own judge model.

LLMJudge
Deterministic fast · cheap

Code-based assertions with no model call — exact/fuzzy matches, schema validity, tool-call and latency/cost checks. Reproducible and free.
